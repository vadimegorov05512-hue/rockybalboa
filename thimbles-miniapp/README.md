# Thimbles Mini App

Небольшая Telegram Mini App игра «Наперстки» без денег и ставок. Игрок следит за шариком, смотрит перемешивание, выбирает наперсток и получает результат. Ведется локальный счет побед, поражений и серии.

## Что внутри

- React + Vite + TypeScript
- Базовая интеграция с Telegram Mini Apps SDK
- 3 наперстка, шарик, перемешивание, выбор, результат
- Локальная статистика: wins, losses, streak, best streak
- Адаптивный интерфейс для мобильного экрана

## Запуск

```bash
cd thimbles-miniapp
npm install
npm run dev
```

Приложение поднимется локально, обычно на `http://localhost:5173`.

## Сборка

```bash
npm run build
npm run preview
```

## Подключение к Telegram Mini App

1. Подними приложение на HTTPS-домене.
2. Создай бота через `@BotFather`.
3. Укажи URL web app для кнопки бота или меню.
4. Открой приложение из Telegram, SDK подхватится автоматически.
